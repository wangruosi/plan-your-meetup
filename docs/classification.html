Topic Classifications¶

By Ruosi Wang 2019/2/11

Data¶

The features were the meetup group descriptions and the labels were the respective topics of the groups. Here is an example of a social event and a tech event.

There are in total of 102691 meetup group descriptions. The median length of the documents of descriptions is 393, with IQR as 463.

Steps¶

  1. Text cleaning was first conducted to remove stop words (e.g., meetup, member, group) and non-word characters (e.g., HTML tags, languages other than English).

  2. Training

    We first split the dataset into a training set (80%) and a testing set (20%)

    Machine learning models: We first extracted the TFIDF (term frequencyinverse document frequency) with chi-square feature selection (top 10000 features). Then, we trained Multinomial Nave Bayes Classifier (MNB) and Linear Support Vector Machine Classifier (SVC) with hyperparameter tuning with cross-validation (20%). We also tested Random Forest Classifier(RFC), and Etra Tree Classifier(ETC) models in the early stage, but they didn't perform very well.

    Deep-learning models: We first tokenized the documents and padded(or trimmed) the sequences to the same length (1200).Then, we employed pre-trained GloVe word vectors as the embedding layer and evaluated the following four structures (common for sequence classification) with cross-validation(20%): 1D convolutional Network (CNN), Gated recurrent unit Network (GRU), and a combination of Con1D and GRU.

  3. Benchmark testing

    To benchmark the performance of these models, we chose the F1-score as the metric, which combined both precision and recall rates and suited better for imbalanced datasets. We examined the classification F1-score on the top 5, 10, 15, and 20 topics with the most number of groups.

The above figure shows the f1-scores of the tested models on 5, 10, 15, and 20 topic classifications.

  • Classification rates decrease as the number of topics to classify increase,
  • GRU and SVC displayed the highest f1-scores.
  • Though MNB is a straightforward model, it performed not much worse than more complicated models and outperformed 1D CNN. One possible reason for this could be that the training sample is still not enough for Deep learning models to gain advantages.

Confusion Matrix¶

vz.plot_conf_matrix('GRU', 10)

Here, we examined the confusion matrix of the top 10 topics (GRU model). The percents results were indicators of classifications. In all off-diagonal cells, the darker the cells, the higher the misclassification rates. The raw number is also shown when a mouse cursor moves over. Besides, the order of topics was sorted by the number of samples (groups) in each class, with career-business being the most and parent-family being the least.

  • Several pairs of topics generated more errors, such as career-business and tech, health-wellness and beliefs, indicating closer relationships with them.
  • The topic tech had the highest recall (89.7%); the topics of social and movements had the lowest recall (70.6%, 70.3%).

Methods¶

The dataset was fetched from the Meetup API utilizing a python API client. In particular, We first identified the Meetup groups(>100 members) in the top 30 US cities with the most number of groups and then gathered the meetup details associated with these groups between 2011 and 2018 July.
